# Step 1.2: Setup shadcn/ui Component Library

**Part of:** PR 1.1 - Next.js Project Setup & UI Framework  
**Focus:** Initialize shadcn/ui and install base UI components  
**Estimated Time:** 10 minutes

## Overview

Set up shadcn/ui component library to provide accessible, customizable UI components built on Radix UI primitives. Install essential components needed for DiveBuddy's interface.

## Pre-Step Checklist

- [ ] **Completed:** Step 1.1 - Next.js project initialized
- [ ] **Running:** Development server can start (`npm run dev`)
- [ ] **Dependencies:** class-variance-authority, clsx, tailwind-merge installed

## Implementation

### Goal

Initialize shadcn/ui and install Button, Card, Input, Badge, and Avatar components.

### Step-by-Step Instructions

#### Step 1: Initialize shadcn/ui

Run the shadcn/ui init command:

```bash
npx shadcn-ui@latest init
```

When prompted, use these settings:
- **TypeScript:** Yes
- **Style:** Default
- **Base color:** Slate
- **CSS variables:** Yes
- **Tailwind config:** tailwind.config.ts
- **Components directory:** @/components
- **Utils directory:** @/lib
- **React Server Components:** Yes
- **Write config:** Yes

This creates `components.json` configuration file.

#### Step 2: Install Essential UI Components

Install the base components needed for DiveBuddy:

```bash
npx shadcn-ui@latest add button
npx shadcn-ui@latest add card
npx shadcn-ui@latest add input
npx shadcn-ui@latest add badge
npx shadcn-ui@latest add avatar
npx shadcn-ui@latest add separator
```

#### Step 3: Verify Installation

- [ ] Check `src/components/ui/` directory exists with installed components:
  - `button.tsx`
  - `card.tsx`
  - `input.tsx`
  - `badge.tsx`
  - `avatar.tsx`
  - `dialog.tsx` (present)
  - `dropdown.tsx` (present)
  - `header.tsx` (present)
  - `footer.tsx` (present)
  - `nav.tsx` (present)

- [ ] Check `src/lib/utils.ts` exists with `cn` utility function:

```typescript
import { type ClassValue, clsx } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| `shadcn-ui: command not found` | Use full npx command: `npx shadcn-ui@latest init` |
| Component installation fails | Check `components.json` exists and is valid JSON |
| TypeScript errors in components | Run `npm install` to ensure all dependencies installed |
| Import path errors | Verify `tsconfig.json` has `"@/*": ["./src/*"]` in paths |
