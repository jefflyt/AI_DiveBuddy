# Step 1.3: Create DiveBuddy Landing Page

**Part of:** PR 1.1 - Next.js Project Setup & UI Framework  
**Focus:** Build landing page with branding, hero section, and navigation  
**Estimated Time:** 20-25 minutes

## Overview

Create a welcoming landing page for DiveBuddy with hero section, feature highlights, and navigation header. This page introduces users to the platform's two core features: diving education and trip planning.

## Pre-Step Checklist

- [ ] **Completed:** Step 1.2 - shadcn/ui components installed
- [ ] **Components Available:** Button, Card, Badge components
- [ ] **Server Running:** Development server accessible at localhost:3000

## Implementation

### Goal
Replace default Next.js landing page with branded DiveBuddy homepage featuring hero section, feature cards, and navigation.

### Step-by-Step Instructions

#### Step 1: Update Tailwind Theme for Diving Colors

#### Step 3: Update Root Layout

Replace `src/app/layout.tsx` with:

```typescript
import type { Metadata } from "next"
import { Inter } from "next/font/google"
import "./globals.css"
import { Header } from "@/components/Header"

const inter = Inter({ subsets: ["latin"] })

  title: "DiveBuddy - AI-Powered Dive Learning & Trip Planning",
  description:
    "Learn scuba diving with AI assistance and plan your next dive trip to Malaysia and Asia Pacific destinations.",
}

  children,

      <body className={inter.className}>
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| `lucide-react` icons not found | Install: `npm install lucide-react` |
| Tailwind classes not applying | Restart dev server: `npm run dev` |
| Layout shifts or spacing issues | Check container and padding classes match |
| Links show 404 | Expected for /learn, /destinations, /chat - we'll create these later |
